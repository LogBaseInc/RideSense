Adding or updating users

Users are what you track with Stick.

  • Users carry a device which is either a physical Stick Device you bought or a Stick Mobile App.
  • For Stick Device, the device ID is shipped along with the device. For Mobile App, this is auto generated.
  • You can choose an Avatar for every user that will be used by live map view.
  • User ID is a user name that you choose to refer back to this user.
