Instructions for Excel Upload

  • Excel file with “.xlsx” extension is supported. No other formats are supported.
  • Please use the template attached here to upload the orders. Download template
  • All the column’s should be as shown in the template.
  • First row should be header.
  • Following fields are mandatory - id, customer name, customer address, mobile number, delivery date, delivery time.
  • Please don’t leave any fields empty. Use “-“ for all missing fields.
  • Tags can have multiple values. Separate the values using “,”
  • Delivery date should be in dd/mm/yyyy format (e.g.: 23/01/2016).
  • Delivery time time should in the format of “start time - end time”. Start time and end time should follow this format hh:mm AM/PM
  • Amount to collect, should be an integer. Don’t use currency symbols.
  • Customer mobile number is 10 digit number.
  • Order number is a unique identification of order. In case if the order number is repeated, order will be updated with the latest information.
  • Only first sheet in excel will be considered. And there should be atleast one order and maximum 100 orders.
